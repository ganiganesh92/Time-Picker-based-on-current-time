jquery.timepicker Based On Current time

A lightweight, customizable javascript timepicker plugin for jQuery inspired by Jon Thornton

In this plugin I am using Jon Thornton timepicker and I have written the timelib-2.0.js library for picking up the time Based on current time

Example

Pass the date time and interval arguments to javascript method 

$("#time").focusin(function () {t('#txtdate','#time',15); });
